# 一、传统法学研究方法

- 关于提出问题、找到争议点的方法；
- 关于问题解构、分析的方法；
- 根据解构出来的问题寻找、收集有关资料的方法；
- 处理、分析收集到的资料的方法；
- 根据资料的分析，对问题的表现、存在与否及原因进行研究的方法；
- 有时还有提出解决问题方案的方法，以及比较或者评价这些方案的方法

# 二、计算法学基本思维方式

## 1.步骤

### 1.1 第一步：先将法学问题形式化为可以通过数据分析予以解决的问题

- 以相关的实际法律数据和对数据的演算规则，即包含数据的算式，来表示法学问题。
- 法学问题的数据化，不是要追求该问题的精确的概念表达，而是为了寻找和发现与该问题研究和解决有关的尽可能全面、完整的数据以及数据分析和计算的可能规则。

### 1.2 第二步：结合计算机技术和系统帮助研究解决数据化后的法学问题

- 分析数据以获得法学问题研究和解决的依据或支撑时，应当遵循两个原则。一是所获得的依据或支撑应当符合法律法规明确的规定，或者法学理论界的学术共识。二是分析数据的算法应当尽可能模拟或接近人为分析数据的过程。
