# ch_1.基本理念

# I.合同的分类
# 1.合同法直接约束
> 《中华人民共和国合同法（1999）》第二条：本法所称合同是平等主体的自然人、法人、其他组织之间设立、变更、终止民事权利义务关系的协议。
婚姻、收养、监护等有关身份关系的协议，适用其他法律的规定。

# 2.参照适用合同法的特殊协议
> 实践中特殊协议主要适用《合同法》总则，但并非绝对。如婚姻家庭协议中涉及到的赠与，参照“赠与合同”的有关规定。

## 2.1 民事合同
> 虽然不由《合同法》直接调整，但仍属于具有法律约束力的协议，受相关民事法律调整。

- 夫妻财产约定；

- 收养协议；

- 离婚协议；

- 遗赠扶养协议；

## 2.2 行政协议
> > 《最高人民法院关于审理行政协议案件若干问题的规定》第二条 公民、法人或者其他组织就下列行政协议提起行政诉讼的，人民法院应当依法受理：
（一）政府特许经营协议；
（二）土地、房屋等征收征用补偿协议；
（三）矿业权等国有自然资源使用权出让协议；
（四）政府投资的保障性住房的租赁、买卖等协议；
（五）符合本规定第一条规定的政府与社会资本合作协议；
（六）其他行政协议。

## 2.3 劳动合同

## 2.4 各类诉讼程序中达成的和解协议、人民调解程序中达成的调解协议
- 如无特殊规定，同样可以适用《合同法》

# 3.参照适用《合同法》
- 特别法对特殊协议有规定的，适用特别法。特别法没有规定的，可参照适用《合同法》；

- 实务中特殊协议主要适用《合同法》总则，但并非绝对[如婚姻家庭类协议中涉及到的赠与，参照分则赠与合同的规定]

# II.合同的表现形式
# 1.广告
> 《最高人民法院关于审理商品房买卖合同纠纷案件适用法律若干问题的解释》第三条：商品房的销售广告和宣传资料为要约邀请，但是出卖人就商品房开发规划范围内的房屋及相关设施所作的说明和允诺具体确定，并对商品房买卖合同的订立以及房屋价格的确定有重大影响的，应当视为要约。该说明和允诺即使未载人商品房买卖合同，亦应当视为合同内容，当事人违反的，应当承担违约责任。

> 《中华人民共和国合同法（1999）》第十五条：【要约邀请】要约邀请是希望他人向自己发出要约的意思表示。寄送的价目表、拍卖公告、招标公告、招股说明书、商业广告等为要约邀请。 
商业广告的内容符合要约规定的，视为要约。

- 虽然一般情况下广告属于要约邀请，但在内容具体确定的情况下，有可能成为合同内容。

# 2.悬赏
> 《最高人民法院关于适用《中华人民共和国合同法》若干问题的解释（二）》第三条：悬赏人以公开方式声明对完成一定行为的人支付报酬，完成特定行为的人请求悬赏人支付报酬的，人民法院依法予以支持。但悬赏有合同法第五十二条规定情形的除外。

# 3.电脑操作、流程、提示

***Q:超市或需要大量采购的生产企业，有自己的供应商管理系统。采购方与供应商在电脑网络系统上下达订单，结算确认。***

- A:该种情况下，电脑网络系统中的操作流程、各种提示确认、通知产生了合同法上的效力。等同于书面合同。

# 4.格式条款
- 停车场门口的告示；

- 商场假一罚十的告示。

# 5.默示行为、不作为
> 《中华人民共和国民法总则》第一百四十条：行为人可以明示或者默示作出意思表示。
沉默只有在有法律规定、当事人约定或者符合当事人之间的交易习惯时，才可以视为意思表示。

- 当事人交易过程中的行为，包括作为和不作为，都有可能产生一定的法律效果。

# 6.协商、沟通过程中的表示
>（2014）三中民（知）初字第13217号：《补充协议二》是否具备了成立要件。原告崴盈公司主张《补充协议二》系双方采用数据电文形式而订立，具备了要约、承诺的成立要件，2013年1月21日被告华谊兄弟公司员工娄睿的邮件为承诺，自该日起《补充协议二》成立。被告华谊兄弟公司则主张本案中原被告双方意欲通过合同书订立合同，签字或盖章才是合同的成立要件。本院认为:

> 第一，合同是当事人意思表示一致的协议。通常情况下，合同的订立为一个动态过程，包括缔约各方接触、洽商、谈判，直至达成合意即合同成立。本案中，原被告双方2012年8月8日、12月10日、12月28日以及2013年1月9日、1月19日的往来邮件，均附有《补充协议二》的不同文本。从形式及内容看，各文本均为合同书形式，且在各文本中，原被告双方对相关条款进行了多次修改，尤其是对其中票房分红基数条款的修改。由此
 可见，双方来往的电子邮件实际是对合同文本的洽商，是合同的洽商、谈判过程。

> 第二，关于涉案影片《西游降魔篇》的发行一事，原被告双方曾先后以合同书形式签订了《合作协议》、《补充协议一》。《补充协议二》的内容亦是关于该影片发行一事，确切而言，是对《合作协议》中第四条发行及收益分配条款的变更，而发行及收益分配条款是事关原被告双方权利义务的重要条款。因此，就《合作协议》中发行及收益分配这一重要条款的变更，以合同书形式进行，更符合双方当事人之间的交易习惯，更能满足交易安全的需要。

> 第三，尽管娄睿2013年1月21日以邮件形式表示对合约基本没有问题。可以随时签字，但该表示应理解为欲以合同书形式订立合同，而非以数据电文形式订立合同的承诺要件。原告崴盈公司亦曾多次催促被告华谊兄弟公司安排签约手续，由此可看出,原告崴盈公司亦有关于签订《补充协议二》书面合同的意向。原告崴盈公司对此解释称，其要求履行签字盖章手续，是根据《中华人民共和国合同法》第三十三条的规定，以数据电文形式订立合同后进行的确认手续。对此，本院认为，该条规定：当事人采用信件、数据电文等形式订立合同的，可以在合同成立之前要求签订确认书。签订确认书时合同成立。即使如原告崴盈公司所述其要求签订的《补充协议二》为确认书，亦应认为《补充协议二》的成立时间为签字或盖章之日，而非2013年1月21日被告华谊兄弟公司娄睿发送邮件的日期。

> 根据《中华人民共和国合同法》第三十二条之规定，采用合同书形式订立合同的，自双方当事人签字或者盖章时合同成立。而本案中，原被告双方对于《补充协议二》均未签字、盖章，不具备成立要件。原告崴盈公司关于《补充协议二》系采用数据电文形式订立，已具备合同成立要件的主张，证据不足，本院不予采信。

## 5.1 合同不成立
- 仅仅构成要约无承诺，或者不构成要约承诺；

- 可能产生缔约过失责任[发生恶意磋商的情形]；

## 5.2 合同成立
## 1）沟通构成要约承诺，成为合同内容
- 双方未签署正式书面合同，沟通中的意思表示产生效力

## 2）沟通不构成要约承诺 + 对合同解释有一定作用 = 间接影响权利义务
> 《合同法》37：采用合同书形式订立合同，在签字或盖章之前，当事人一方已经履行主要义务，对方接受的，该合同成立。

- 双方未签署正式书面合同 + 一方履行主要义务，对方接受 = 合同成立，双方就合同的谈判沟通内容成为合同内容的参考依据。

## 3）合同成立后的沟通可能构成新的要约 = 承诺，变更、解除原合同，成立新合同
- 双方签署正式书面合同 + 但对合同中特定权利义务约定不明作出进一步补偿或解释，双方就该交易的沟通内容，具有参考意义；

## 4）合同成立后的沟通作为实施行为 = 合同全部或部分履行、违约、合同解除等效果
- 双方签署正式书面合同 + 合同履行过程中通过函件、沟通对合同的权利义务进行变更、补充、通知解除等；

- 如通知解除合同、未按照约定提交资料；

## 5.3 沟通内容构成侵权
- 泄密；

- 侵犯知识产权；

- 侮辱诽谤；

- 广告违反广告法；

# III.合同起草、修订的原则
# 1.尽量促成交易，不过分追求合同文本的“完美”
## 1.1 有争议条款
- 不重要的应适当让步，避免因为不重要的分歧点而影响交易进行；

- 重要条款必须坚持，同时应适当批注、说明向对方解释，为什么必须坚持，请对方予以理解。

## 1.2 对方提供的格式文本
- 无需在不涉及实质权利义务的合同格式、措辞、辅助条款上进行修改；

## 1.3 简单小额的交易
- 简便的合同文本有利于双方迅速达成一致和签署。

## 1.4 对细节条款尚未达成一致
- 此时为了尽快推动交易，可以确认主要条款，细节问题可以留待日后解决；

# 2.合同设置的交易流程应当适当简便，便于理解和执行
***Q：建筑公司拟与供应商签订长期采购合同，约定以现场签售作为结算货款依据。长期采购合同中应设置谁作为现场签字确认人员？***

- A:由财务负责人签字，或由2名指定人员共同签收确认。项目经理未必每天都在工地，无法签收会影响采购；随便1个员工签收会出现串通损害公司利益的风险。

# 3.商业因素的考量
***Q:A企业生产的汽车零部件在业内受到好评，但A坚持要求在采购条款中约定：A企业承担的赔偿责任（无论是否是因为质量原因或何种责任），不超过货物金额的200%，该条款是否可接受？***

- A: depends 该零部件的具体使用场景：1）无关人身安全，如车内空调、装饰、音箱零部件，该条款可接受；2）如果是导致事故的零部件，如发动机、轴承等，该条款可要求修改（如对责任限制条款增加例外情形）。

# IV.实务中的考虑要点
# 1.负面与风险
## 1.1 价款、费用条款
- 正常的费用标准；

- 合同提前解除、非正常履行时，费用如何结算；

## 1.2 期限
- 合同提前解除或终止，如何处理；

- 逾期的处理

## 1.3 义务的履行/违约责任
- 如果不能正常履行如何设置违约救济措施；

- 法律规定的违约责任可不必在合同中说明，需要针对的是“如果没有特别责任约定，导致我方难以追究对方责任，我方损失很大”

## 1.4 违约责任救济措施
***Q:租赁合同中，要求合同解除终止后，承租方在3日内搬出并退房。如果承租方不搬走？***

- A:规定救济措施：如果承租方拒绝退房，则应按照原租金标准的150%支付房屋占用费，直至退房之日为止。

# 2.重要环节的取证
> 《最高人民法院关于民事诉讼证据的若干规定(2019修正)》

## 2.1 交货
- 收货人；

- 签收方式[签字或盖章]；

- 长期合同下有多个订单，规定订单经双方签字盖章生效有难度、较为繁琐[大型公司盖章需要审批]时，可以改为在主交易合同（经双方盖章）中指定电子预想来确认订单的方式。

## 2.2 提交工作文档、工作成果、智力成果
- 对方接收的方式；[电子邮箱]

## 2.3 验收、确认
- 货物质量标准、工作成果的验收标准；

- 提出异议的方式[一定期限内未验收视为通过验收]；

## 2.4 费用结算
- 费用的凭据；

- 对账结算方式；

## 2.5 通知
- 联系方式[避免对方否认收到通知]；

- 履行周期较长的合同，必须明确联系方式。

## 2.6 收款付款
- 以银行转账方式，无需再要求收款方出具收据；

- 收款账号并非对方公司账号、对方本人账号，需在合同中专门列明对方的制定收款账号；

- 以现金支付，收款同时必须出具对方本人签字或本单位盖章（公章、财务专用章）的收据；

## 2.7 披露义务
- 标的各类瑕疵[货物瑕疵，资产上已经设定的担保、原有债务]；

- 披露义务与否，可能直接导致索赔、解除合同等，必须在合同或合同附件中以明确方式予以披露、声明。

## 2.8 证明对方违约
- 对违约方的积极违约行为，即作出合同中明确禁止的行为[超出经营范围、超出范围使用、违反竞业限制]

### 1）改变证明标准
- 约定只要出现某些情形的任意一种，即视为员工违反了竞业限制义务。

- 如，在竞争对手处缴纳社会保险、直系亲属或配偶在竞争对手处领取费用而无合理说明；

## 2）要求对方适当举证
- 约定对方有义务定期提交就业情况说明，如未提供，则我方有权暂停支付竞业限制补偿；

## 2.9 已经发生的法律事实在合同中固定
- 合同或文件签署时，已经发生的重要法律事实在合同、文件中予以明确。[已经履行的权利义务、已经支付的款项、已经出现的违约事实等]
