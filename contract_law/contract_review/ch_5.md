# I.意向书
> 当事人无意订立正式合同的情况。

# 1.表现形式
## 1.1 意向协议、框架协议
- [12322]投资意向书；

- [3373]项目合作意向书；

## 1.2 TS/Term Sheet/交易条款清单
- 在PE/VA 、M&A 中广泛运用；

## 1.3 谅解备忘录/memorandum of understanding
- 作为法律文件或商务文件的备忘录，主要内容是各方对后续交易达成的初步一致意向。

- [8328]上市公司并购交易备忘录；

## 1.4 会议纪要
- usu为双方但是人就某一问题谈判、磋商的文件，用于记录谈判、会议的主要内容‘

- incl政府会议纪要，即政府派员参与当事人之间经济利益的协调，对当事人之间的民事行为提出的一种协调意见和协调过程的记录。

# 2.条款结构
## 2.1 首部
- 文件名称；[认购书、订购书、确认函实践中多为预约合同；协议、合同容易引起争议]

- 当事方；

- 签署时间地点；

## 2.2 鉴于
- 签署意向书的背景，可不写；

## 2.3 交易条款部分
> 对于意向书而言，“交易条款部分”是无法律效力的，如果有意让其生效。或部分生效，则不应采用意向书的形式；

- 达成一致的事项；

- 尚需进一步明确、磋商的事项；

- 下一阶段的行动安排；

## 2.4 通用条款部分
### 1）主要法律条款
- 法律效力；

- 保密；

- 排他期/独家谈判权；

- 争议解决/法律适用、管辖；

### 2）可包含内容
- 费用各自负担；

- 不得转让；

- 通知送达；

- 语言；

- 定义；

## 2.5 签署页
- 意向书的签字盖章处；

- 如有类似于“本文件自各方签署后生效”的表述应当删除，否则与意向书效力矛盾；

## 2.6 附录及附件
- 附带的背景法律文件；

- 可没有；

# 3.意向书审查要点
## 3.1 法律效力条款
> 该条款通过有线列举的方式，明确约定整个意向书中哪些条款有效，除此之外均无效

### 1)明确约定整个文件均无法律效力
- 此情况为文件内容中只有交易条款部分，无通用条款；

### 2）明确部分条款具有法律效力，其余无法律效力
- 实务中的通常做法；

- 注意不要把“交易条款部分“列入有效条款中；

## 3.2 保密
- 在意向书中明确对“本意向书内容及本意向书存在的事实”进行保密；[标的金额较大的交易，esp涉及上市公司，双方不希望对外披露正在进行此交易的事实]

- 还会约定一般的商业秘密保密义务；

## 3.3 排他期
- usu 在投融资领域的交易中明确约定，要求潜在接受投资方在一定期限内不得与其他意向投资方接触、谈判；

- 排他期通常为60-90days，一般不会更长；

- 若一方不接受，可以删除；

## 3.4 赔偿损失/违约责任
- 一般无法律效力，只能约定赔偿损失；

- 违反有效条款时会构成违约责任；若是约定违约责任，应将该条款也纳入“具有法律效力的条款”范围之内；

## 3.5 实质权利义务内容
### 1）常见的实质权利义务条款
- 费用支付；

- 利益分配机制；

- 知识产权许可或转让；

- 独家或排他性权利/独家合作；

### 2）无权利义务内容意向书的审查
- 无需管辖、不可抗力、可分性、不得转让条款、违约责任；

- 无需过多修改合同条款，保密条款只要是常规、合理，可接受；

- 提示客户“合同可签署，但缺少具体的、实质的权利义务，仅为初步框架协议，难以约束对方；进入具体交易时，需另外签订正式合同。”

### 2)缺少实质权利义务的战略合作协议
- 说明各方拟加强合作，大致的合作目的、合作内容；

- 如在宣传上互相列为合作机构等，或政府与企业间签订，或企业与企业间签订；

### 3）无偿、非经营性的合作协议
- 合同中缺少实质权利义务条款，导致在事实上难以要求对方承担什么责任；

# II.预约合同
# 1.概念
> 《中华人民共和国民法典（2020）》第四百九十五条：当事人约定在将来一定期限内订立合同的认购书、订购书、预订书等，构成预约合同。 
当事人一方不履行预约合同约定的订立合同义务的，对方可以请求其承担预约合同的违约责任。 

> 《最高人民法院关于审理买卖合同纠纷案件适用法律问题的解释》第二条：当事人签订认购书、订购书、预订书、意向书、备忘录等预约合同，约定在将来一定期限内订立买卖合同，一方不履行订立买卖合同的义务，对方请求其承担预约合同违约责任或者要求解除预约合同并主张损害赔偿的，人民法院应予支持。

# 2.表现形式
- 订购数、认购书、预定书、认购协议；

- 意向书、备忘录；

- 定金合同；[如果对方拒绝签订、履行后面的正式合同，则应承担定金责任]

- 战略合作协议；[此种战略合作协议的内容是双方准备签订具体交易的合同]

# 4.意向书、预约合同、本约合同 diff
|      |  意向书 |  预约合同  |  本约合同 |
| ---- | ----- | ---- | ---- |
| 所在阶段 | 要约承诺之前的磋商阶段 | 本约磋商过程中，确定在某一时刻订立本约；但已经完成了预约合同本身的要约与承诺 |  本约合同已经成立 |
|  性质  |  表达交易意愿的磋商性文件，不具有合同的法律效力 |  合同 | 合同 |
|  目的  |  表达交易愿意，继续磋商 | 在一定期限、条件下订立本约 | 建立合同关系，履行具体权利义务，实现合同目的  |
|  确定性  |  交易内容、是否交易不确定 |  交易对象、订立本约已经确定；其他内容不明确   | 合同内容、各方权利义务已经确定 | 
|  法律约束力  |  无  |  有法律约束 + 本约成立即终止；某些情况下，预约合同的内容会成为本约的一部分 |  有 | 
|  义务 | 诚信磋商 |  在确定时间订立本约  |  履行完毕所有合同内容义务 | 
|  责任承担  |  缔约过失  | 违约责任，但有特殊性 |  违约责任，可要求继续履行，赔偿可期待利益 |

# 5.起草要点
## 5.1 是否起草预约合同
- 如果已经符合签订本约合同的条件，则应签署本约合同；只有确实无法确定本约合同的具体条件，才签署预约合同；

- 预约合同的应用场景：当事人意图签订本约，但由于某种原因无法签订；只有主要条款已经商定，还有部分条款未谈妥；交易意图尚不明确，双方处于“观望期”，需要给予对方一定选择空间；

- 实务中预约合同主要存在于不动产买卖（包括一手房、二手房）、土地使用权转让、房屋租赁等领域中。

- 如果是需要审批才生效的交易，仍然可以签署本约，只要合同要待审批通过后方可生效；

## 5.2 正式合同 + 条件安排
> 若我方或对方希望保留合同生效或解除、变更的灵活性，也不一定要签订预约合同，而可直接签订正常合同，同时在合同中对合同生效条件、解除、变更条件作灵活约定；

### 1）附条件生效的合同
***Exp1:受让方在银行的用信额度恢复到10亿元”***

***Exp2:转让方股东大会批准本次转让。***

- 一旦条件成就，合同即生效；无需再专门签订合同；

### 2）附解除或终止条件的合同
- 约定符合条件时，一方或双方有权解除合同；

- 应注意此种解除合同下的责任，如明确约定不承担违约责任或承担优先的违约责任，对方不得再要求继续履行；

### 3）附变更、选择条件的本约合同
***Exp:如条件成就，就履行其中的A关系；如条件不成就，就履行其中B关系。***

- 实务中这种做法较少；

# 6.审查要点
## 6.1 特征条款
- 双方在一定期限或条件下应签订本约，如果在期限内或条件符合时拒绝签订本约合同即为违约；

- 本约合同的主要条款，一般是标的、价款——如果拒绝按该约定签订本约即为违约，有时还会加上“独占协商条款”，类似于“在乙方按照本条约定签订《商品房销售合同》之前，甲方不得将该房另售他人”，相当于从反面要求对方必须和我方签订本约。

## 6.2 拒签本约承担的违约责任
### 1）单务预约/一方预约
***Exp）乙方有意承租甲方商铺，但乙方不确定自己公司总部允许乙方加盟及允许开店。双方签订预约合同，约定一个月内签订正式租赁合同。2）若甲方与他人签约，甲方承担违约责任；3）一方为能与总部签订特许经营合同时，则可以不签约也不承担违约责任；***

- 一方可拒绝签订本约 + 无需承担违约责任；

### 2）一方违约 + 承担定金罚则
***Exp1:根据法律规定，定金在实际支付后方可发生定金效力。***

***Exp2:商品房认购中，可约定如因限购、无法取得银行贷款导致无法购买的，不视为违约，不承担定金责任。***

- 合同中需列明“不签约本约不视为违约”；

- 如果希望强化定金责任，则应明确本约签订的期限，尽可能列出本约合同的条款，否则任何一方可能以本约合同的任何一个条款无法达成一致为由决绝签订本约。

### 3）一方违约 + 承担违约责任
***Exp:承担约定的违约责任之后，不再承担其他的违约责任（其他损失或可得利益损失），对方也不能要求继续履行。***

- 实务中还需在预约合同中明确约定“违约责任的限制”；

- 如果我方有意要求对方更多的赔偿，则可以加点定金责任、违约责任；

- 如果我方有意要去对方继续履行，则应修改为正式的本约合同，不再使用预约合同。

## 6.3 预约合同与本约合同的衔接
***Exp:在本约合同中写明：双方于X年X月X日签订的认购书/预约合同于本合同签署后失效。***

# III.格式条款、格式合同
# 1.概念
> 《中华人民共和国民法典（2020）》第四百九十六条：格式条款是当事人为了重复使用而预先拟定，并在订立合同时未与对方协商的条款。 
采用格式条款订立合同的，提供格式条款的一方应当遵循公平原则确定当事人之间的权利和义务，并采取合理的方式提示对方注意免除或者减轻其责任等与对方有重大利害关系的条款，按照对方的要求，对该条款予以说明。提供格式条款的一方未履行提示或者说明义务，致使对方没有注意或者理解与其有重大利害关系的条款的，对方可以主张该条款不成为合同的内容。 

> 《中华人民共和国民法典（2020）》第四百九十七条：有下列情形之一的，该格式条款无效：
（一）具有本法第一编第六章第三节和本法第五百零六条规定的无效情形； 
（二）提供格式条款一方不合理地免除或者减轻其责任、加重对方责任、限制对方主要权利； 
（三）提供格式条款一方排除对方主要权利。 

- 如果是加重己方的责任，并不受上述规定限制；如无其他无效事由，应该有效。

> 《中华人民共和国民法典（2020）》第四百九十八条 对格式条款的理解发生争议的，应当按照通常理解予以解释。对格式条款有两种以上解释的，应当作出不利于提供格式条款一方的解释。格式条款和非格式条款不一致的，应当采用非格式条款。

> 《最高人民法院关于适用《中华人民共和国民事诉讼法》的解释》第三十一条 经营者使用格式条款与消费者订立管辖协议，未采取合理方式提请消费者注意，消费者主张管辖协议无效的，人民法院应予支持。

> 《合同违法行为监督处理办法》第九条：经营者与消费者采用格式条款订立合同的，经营者不得在格式条款中免除自己的下列责任： 
（一）造成消费者人身伤害的责任；
（二）因故意或者重大过失造成消费者财产损失的责任； 
（三）对提供的商品或者服务依法应当承担的保证责任； 
（四）因违约依法应当承担的违约责任； 
（五）依法应当承担的其他责任。 

> 《合同违法行为监督处理办法》第十条：经营者与消费者采用格式条款订立合同的，经营者不得在格式条款中加重消费者下列责任： 
（一）违约金或者损害赔偿金超过法定数额或者合理数额；
（二）承担应当由格式条款提供方承担的经营风险责任； 
（三）其他依照法律法规不应由消费者承担的责任。 

> 《合同违法行为监督处理办法》第十一条：经营者与消费者采用格式条款订立合同的，经营者不得在格式条款中排除消费者下列权利： 
（一）依法变更或者解除合同的权利； 
（二）请求支付违约金的权利； 
（三）请求损害赔偿的权利；
（四）解释格式条款的权利； 
（五）就格式条款争议提起诉讼的权利； 
（六）消费者依法应当享有的其他权利。 

> 《侵害消费者权益行为处罚办法》第十二条：经营者向消费者提供商品或者服务使用格式条款、通知、声明、店堂告示等的，应当以显著方式提请消费者注意与消费者有重大利害关系的内容，并按照消费者的要求予以说明，不得作出含有下列内容的规定：
（一）免除或者部分免除经营者对其所提供的商品或者服务应当承担的修理、重作、更换、退货、补足商品数量、退还货款和服务费用、赔偿损失等责任； 
（二）排除或者限制消费者提出修理、更换、退货、赔偿损失以及获得违约金和其他合理赔偿的权利； 
（三）排除或者限制消费者依法投诉、举报、提起诉讼的权利；
（四）强制或者变相强制消费者购买和使用其提供的或者其指定的经营者提供的商品或者服务，对不接受其不合理条件的消费者拒绝提供相应商品或者服务，或者提高收费标准；
（五）规定经营者有权任意变更或者解除合同，限制消费者依法变更或者解除合同权利； 
（六）规定经营者单方享有解释权或者最终解释权； 
（七）其他对消费者不公平、不合理的规定。

# 3.格式条款审查注意事项
## 3.1 重点条款
### 1）内容
- 要求对方承担违约金、赔偿责任、增加费用的；

- 要求对方不得解除或终止合同，不得要求退款的扣除相当一部分费用的；

- 说明我方不承担责任，或限制我方承担责任的；

### 2）对重点条款说明、特别提示
- 字体加大加粗、特殊颜色，与一般条款相区别；

- 对方签约处专门增加签约提示条款。签约提示条款本身也应加大加粗；

- 让对方抄写指定文字 + 签署；位置一般在尾部签约处；

- 除了正常签署合同外，就重点条款部分另做专门一页，由对方签署确认；

- 网站上的格式条款，要求对方专门点击确认该条款；

- 以张贴、告示体现的格式条款，应该注意张贴在比较明显的位置，保证向对方能够看到/足以让法官认定为“知道或应当知道”；

- 张贴的当时进行拍照、录像、存档；可证明张贴的时间；

## 3.2 经营者与消费者之间格式条款处罚
### 1）订约自由
***Exp:此卡售出，概不退换。***

***Exp：商品售出，概不退换。***

***Exp:打折商品，不退不换。***
