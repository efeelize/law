# ch_1 执行基本问题
# I.执行依据
# 1.执行结案通知书不具有裁定的效力
- 执行案件结案通知书作为执行案件终结的告知类法律文书，它的主要作用在于告知双方当事人该起案件已经执行完毕，其是在经过执行阶段的查封、扣押、拍卖等程序后，财产关系等已经处理完毕后作出的程序性告知，该通知书本身不具备确定申请人与被执行人之间权利义务、财产关系的属性。









# II.执行对象：被执行人的财产
# 1.个体工商户
## 1.1 法律特征
> 《个体工商户条例》 第二条：有经营能力的公民，依照本条例规定经工商行政管理部门登记，从事工商业经营的，为个体工商户。
个体工商户可以个人经营，也可以家庭经营。
个体工商户的合法权益受法律保护，任何单位和个人不得侵害。

> 《中华人民共和国民法总则》 第五十四条：自然人从事工商业经营，经依法登记，为个体工商户。个体工商户可以起字号。

- 1）从事工商个体经营的是单个自然人或者家庭，单个自然人申请的，必须是具有民事行为能力的自然人；

- 2）个体工商户必须依法进行核准登记，自然人或者家庭进行个体经营工商业，必须依法向工商行政管理部门提出申请，经核准登记，颁发个体经营的营业执照后，才取得个体工商户资格，且必须依法在核准登记的范围内从事经营活动；

- 3）个体工商户是在法律允许的范围内从事工商业经营活动。该工商业经营活动应做广义理解，除手工业、加工业、零售业以外，还包括修理业、服务业等；

- 4）但是不论进行何种工商业经营活动都应在法律允许的范围内进行，方可以受到法律保护。

## 1.2 财产
> 《中华人民共和国民法通则（2009）》 第二十九条：个体工商户，农村承包经营户的债务，个人经营的，以个人财产承担；家庭经营的，以家庭财产承担。

- 1）基于个体工商户的法律特征，其财产与经营者的自然人个人或家庭财产不可分离。无论从经营资金的来源还是经营收入的用于均与经营者个人及家庭具有融合性；

- 2）个体工商户与一人公司相区别，个体工商户的字号和财产作为经营者的个人信息、个人财产具有高度的重合性和对应性，法律规定根据个体工商户按自然人经营和家庭经营的方式，区分其实以经营者的自然人个人全部财产承担或是以家庭全部财产承担无限责任的非法人组织。

## 1.3 个体工商户和实际经营者
> 《最高人民法院关于适用《中华人民共和国民事诉讼法》的解释》 第五十九条：在诉讼中，个体工商户以营业执照上登记的经营者为当事人。有字号的，以营业执照上登记的字号为当事人，但应同时注明该字号经营者的基本信息。
营业执照上登记的经营者与实际经营者不一致的，以登记的经营者和实际经营者为共同诉讼人。

- 1）实践中，有些有字号的个体工商户将执照转包或出租给他人经营适用，或者以更换字号的方式，将其财产全部登记在他人名下等逃避债务及法律责任；

- 2）如果仅将个体工商户作为诉讼主体和执行主体，没有列明实际经营者的信息，就可能使得执行落空。因为现有的国家行政管理规定，私人房屋、车辆等不动产和动产是登记于自然人名下的，其存款也是自然人名下的。

- 3）个体工商户虽然相对作为实际经营者的自然人有独立性，但不是绝对独立的，该种相对独立性仅体现于经营者的变化一般不影响其法律人格的变化；

# IV.执行时效
> 《最高人民法院关于适用《中华人民共和国民事诉讼法》的解释》第四百八十三条：申请执行人超过申请执行时效期间向人民法院申请强制执行的，人民法院应予受理。被执行人对申请执行时效期间提出异议，人民法院经审查异议成立的，裁定不予执行。 
被执行人履行全部或者部分义务后，又以不知道申请执行时效期间届满为由请求执行回转的，人民法院不予支持。

- 1）执行时效的设置目的是督促权利人尽快主张实体权利，积极主动实现债权。由于执行时效具有程序性权利性质，申请执行时效期间已经经过，申请执行人即丧失请求法院强制执行其债权的权利。

- 2）但经执行依据所确认的双方当事人之间的实体权利并不消灭，若被执行人的异议理由成立，则法院应当对这一程序性权利作出评价，而不应对实体权利作出处理。即执行裁定书裁决事项应为对法院作出强制执行通知书不予执行，而不是对裁决依据本身不予执行。

# 1.中断
> 《最高人民法院关于适用《中华人民共和国民事诉讼法》执行程序若干问题的解释》第二十八条：申请执行时效因申请执行、当事人双方达成和解协议、当事人一方提出履行要求或者同意履行义务而中断。从中断时起，申请执行时效期间重新计算。

> 山东省高院（2018）鲁执复34号执行裁定书：

- 1）作为仲裁裁决书确认申请执行人A与被执行人B签订的《债权转让协议》有效，B需要向A履行支付债权转让款的义务，仲裁裁决为（2014）中国贸仲京裁字第0786号裁决书；A对0786号裁决不服，向北京市四中院申请撤销仲裁裁决，四中院驳回撤销0786号仲裁裁决书的申请；A向中国贸仲再次提出仲裁申请，请求解除双方签订的《债权转让协议》，贸仲作出（2017）中国贸仲京裁字第0870号裁决书，驳回A的请求；2017年，A向法院申请强制执行第0786号裁决书。

- 2）A向贸仲提出撤销仲裁申请、提出新的仲裁请求，但这2个申请内容并非对0786号仲裁裁决所确认的债权主张权利，与申请执行不具有同等导致执行时效中断的效力；其本意是不想实现债权，亦不想申请实现债权，并想达到解除《债权转让协议》的目的。因此，上述2行为不能引起申请执行时效的中断。

- 3）B公司在2017年A公司申请仲裁庭审过程中对履行义务作出了承诺，但0786号裁决书于2014年作出且送达双方当事人。B作出履行承诺时，申请执行时效期间已经届满；能够产生申请执行时效中断的履行义务的意思表示应当在申请执行期间内作出。期间届满后做出的承诺不能产生申请执行时效中断的结果。
