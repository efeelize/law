# 一、集装箱运输设备
> 标准箱（ETU,twenty-foot equivalent unit）;40ft 集装箱 == 2TEU

> 40ft 集装箱为FEU(forty-foot equivalent unit);未经换算的集装箱称为自然箱（unit）

## 1、普通货物集装箱/干货集装箱  
|  集装箱种类   |    货物分类   |   使用  |
|  ------- | ------ | ------|
|  杂货集装箱 |   干杂货  |  使用时应当注意最大负荷  |
|  敞顶集装箱/开顶箱  |    玻璃板、钢铁制品、机械设备等高度较高的货物   |  使用起重机将重货从顶部装入集装箱内，或从顶部将货物吊出。将货物装入后，再用防水布遮盖顶部，以防货物受损  |
|   台式集装箱   |  不适于装入杂货集装箱和敞顶集装箱的长大件、超重件、轻泡货、重型机械、钢材、钢管、裸装设备  |  没有箱顶和箱壁，箱端壁也可以卸下，只留下箱底和四角柱来承受货载的集装箱  | 

## 2、保温集装箱
|  集装箱种类   |    货物分类   |   使用  |
|  ------- | ------ | ------|
|  内置式机械冷藏箱  |  冷冻食品  |  箱内附有制冷机  |
|   外置式机械冷藏箱  |  冷冻食品  |  箱内未附有制冷机，只是在前段设有冷气吸入口和排气口，由船上制冷装置和固定管路供应冷气  |
|  隔热集装箱  |  水果、蔬菜  |  通常用干冰制冷   | 
|  通风集装箱  |  不需要冷冻，且具有呼吸作用的水果蔬菜  |  端壁上开有通风口  |

## 3、特种集装箱 
|  集装箱种类   |    货物分类   |   使用  |
|  ------- | ------ | ------|
| 散货集装箱 | 大豆、大米、麦芽、面粉、饲料、水泥、化学制品  | usu铝制、钢制。为提高卸货时的滑溜效果，箱内底板大多用玻璃钢，侧壁内衬一般为刨平的木板。箱底有的做成漏斗形状的，有的从箱门处流出卸货 。若是有在港外锚地进行熏蒸消毒的要求的，要求在散货集装箱上设置投入熏蒸药物的开口，以及熏蒸气体排出口，并且要求这种集装箱在熏蒸时能完全气密 |
|  罐状集装箱  |  酒类、油类、化学品 |  装货时，货物由液罐的顶部装货孔进入；卸货时，货物由排出孔靠重力作用自行流出，或由顶部装货孔吸出  |

# 二、按照货物性质对集装箱进行分类

|  ------- | ------ | ------|
| 清洁货物  | 本身清洁、干燥、对积载和保管没有特殊要求，与其他货物混载不会使其他货物受损的货物 | 纺织品、纺织制品  |
| 污秽货物 |  本身可能渗出汁液、杨散、散落、散发恶臭等性质，与其他货物混载，可能使其他货物发生湿损、串味、虫蛀等损害的货物  | 水泥、生皮、碳黑、油脂、沥青、椰壳、牛骨、干生皮、胡椒 |

## 4、集装箱的积载
> 以堆放一个集装箱为单位，以20ft集装箱的箱位编号为例：
- 1）跨位(bay)：从船首到船尾，01；03；05（40ft为02；04；06）
- 2）幅位（row）:以船舶中部为界限，左舷自右向左为02；04；06；右舷自左向右为01；03；05
- 3）层次（tier）:自船底自下而上：X2;X4;X6
- like:030206--20ft集装箱堆放于第二跨位、坐舷一侧舱内第一幅位的第三层位置上

# 三、集装箱数量的确定
> 所谓轻货、重货是相对于集装箱的单位容积能承载的重量而言的。
## 1、集装箱货数据
| 集装箱种类 | 最大载货量 |  集装箱容积：箱容利用率为100%  |  集装箱容积：箱容利用率为80%   |  单位容量：箱容利用率为100%  |   单位容量：箱容利用率为80%  |
| 20ft杂货箱 |  21790  |  33.2  | 26.6 |  656.3  | 820.4  |
| 40ft杂货箱 |  27630 |  67.8 | 54.2  |  407.5 |  509.4  |
| 20ft敞顶箱 |  21480  |  28.4  | 22.7  |  756.3  |  945.4  |
| 20ft台架箱 |  21230  |  28.5  |  22.8  | 744.9  |  931.1  |
 
## 2、公式
> 集装箱单位容量==集装箱最大载货量/集装箱容积
> 载重==总重-自重
> 箱数==货物重量/集装箱载量
> 集装箱内有效内容积==内容积* 有效利用率

- if 货物密度200kg/m^2 < 509.4kg/m^3;else 轻货，适用40ft杂货箱 
- if 货物密度700kg/m^2 ; 509.4kg/m^3<700kg/m^3<820.4kg/m^3 ;else 重货，适用20ft杂货箱
- if 货物密度>820.4kg/m^2 ;else 高密度或，使用20ft杂货箱时应防止超重

## 3、例题
### 3.1 一批出口机电产品共60件，小木箱包装，毛重36060kg,体积28m^3。已知按80%箱容利用率计算的单位容量20ft为820.4kg/m^3\40ft为509.4kg/m^3;集装箱的内容积20ft为33.2m^3、40ft为67.8m^3;集装箱的自重是：20ft杂货箱2210kg、40ft杂货箱2850kg、20ft敞顶箱2520kg、20ft台架箱2770kg,请选择最佳集装箱种类并计算所需箱量
- 1）货物基本情况：该产品货物密度为 36060/28==1288kg/m^3;1288>820>509;属于重货，选用20ft杂货集装箱
- 2）载重：24000-2210=21790kg
- 3) 所需集装箱数：36060/21790==1.65
- 4) so:需要2个20ft杂货集装箱

### 3.2 一批出口货物，其中有不同名称、规格、型号的工具共450箱，毛重30512kg,体积32m^3
- 1)货物密度：30512/32==953kg/m^3;重货
- 2）载重：24000-2210==21790kg
- 3) 箱数：30512/21790==1.4
- 4）so:需要2个20ft杂货集装箱

### 3.3 一批纺织物出口，其中：童装短裤50箱，毛重612kg;女士长裤75箱，毛重1350kg;男士上装200箱；毛重1540kg;男士短裤525箱，毛重4940kg.总体积为52m^3
- 1)货物总重量：612+1350+1540+4940==8442kg;
- 2)货物密度：8442/52==162.35kg/m^3;162.35<509<820;低密度货物；属于轻泡货
- 3）选用40ft杂货集装箱的有效内容积：67.8* 80%==54.24m^3
- 4）箱子数量：52/54.24==0.96
- 5) so:需要1个40ft杂货集装箱

# 四、集装箱的流通途径
## 1 支线运输
> 为了减少集装箱船的挂靠港，通过陆路或海路向集装箱枢纽港集中装箱的运输

## 2 干线运输
> 在各设有集装箱码头的枢纽港之间，载运集装箱的跨越海洋的运输
### 2.1 广义
           就集装箱多式联运而言，从起运地通过陆路向枢纽港集中集装箱的运输||从枢纽港向内陆疏云集装箱的陆路运输
### 2.2 狭义
           only 从附近的港口向集装箱枢纽港集中装箱||反向的运输

# 六、集装箱码头
## 1 功能
- 1.1 接受/发送由铁路||公路运来的集装箱；
- 1.2 向/从集装箱船装/卸集装箱

## 2 主要设施
### 2.1 泊位
        ie.在码头内，给船舶停靠的岸壁线与对应水域构成的区域
           泊位的长度、水深depends港口类型、码头种类和将要停靠的船舶的种类、大小->
           泊位长度应大于集装箱船的全长
           
### 2.2 码头前沿
        ie.码头壁岸线，从泊位案壁到堆场前（防汛墙）之间的码头区域
           安装大型岸边装卸桥，进行集装箱船岸之间的装卸；
           usu 30-60m,以满足对方船舱盖板和保证足够的陆运机械通道，确保集装箱装卸桥的作业效率

### 2.3 集装箱堆场
> 前后并没有严格的划分，only 地理位置上的相对概念
#### 1）前方堆场/出口箱区、临时堆场、过渡堆场、编排场、调度场
        ie.码头前沿和后方堆场之间，为加快船舶装卸作业效率，用于临时堆放集装箱。
        used for:船舶到港前，预先堆放将装船出口的集装箱并按装船次序进行编排；卸船时，临时堆存卸船进口的集装箱。
#### 2）后方堆场
        ie.贮存和保管空、重箱的场地，是码头堆场中除前方堆场以外的部分
        incl:中转箱堆场、进口重箱堆场、空箱堆场、冷藏箱堆场、危险品箱堆场
#### 3）in practice
        1)堆场前方：出口箱
        2）中间：中转箱
        3）后方：进口箱、冷藏箱、危险品箱、空箱
#### 4）编号：
        1）场箱位
           以20ft集装箱箱底为标准大小的长方形格子
        2）箱区号
           每个箱区的两端，都有对应的编号
        3）场箱位号
           堆存在堆场上的集装箱，其位置都有唯一的编码标识

### 2.4 集装箱货运站
#### 1）主要业务
- 1）拼箱货的理货和交接
- 2）检验货物的外表状况、验收货物，并签发场站收据，发现货物有异状时，在场站收据上做出相应的批注；
- 3）拼箱货的配箱、积载、和装箱
- 4）联系海关监装
- 5）货物装箱后在箱门上施加铅封
- 6）进口货箱的拆箱、卸货、保管和交付
- 7）办理各项单证的编制、签发和流转
- 8）向有关方结算费用

### 2.5 控制室/控制中心/控制塔/指挥塔
        ie.集装箱码头各项作业的指挥调度中心
        incl:电子计算机系统、测风仪、气象预报系统、VHF（无线对讲机）、CCTV、望远镜
        
### 2.6 检查口/检查桥/闸口/大门
        ie.集装箱码头的出入口，集装箱和集装箱货物交接点，也是区分码头内外责任的分界点
        cuz:1)道口既要检查和交接集装箱及其货物的有关单证；
            2）还要确认交接的事实和交付的对象，办理集装箱和货物的交接责任，检查有关箱号、铅封号、集装箱箱体和货物的外表状况等
            
### 2.7 维修车间
        ie.对集装箱和集装箱专用机械进行检查、修理和保养的地方
        usu 在码头后方或在保养区的附近

## 6.3 主要业务
- 1）对整箱货（包括由集装箱货运站完成拼装的整箱）进行交接、存贮和保管；
- 2）安排集装箱船的靠泊、装卸和编制积载计划；
- 3）编制有关货运单证；
- 4）编制并签署集装箱及运载工具的出入及流转单证；
- 5）联系及办理集装箱及运载工具、装卸机械的检查、维修及空箱的清扫、重箱的熏蒸消毒
- 6）空箱的收、发、贮存和保管
- 7）安排空、重箱在场地的堆码和编制场地分配计划
- 8）提供冷藏集装箱所需要的电源
- 9）向船公司报告到港集装箱船的动态
- 10）向有官方结算费用
